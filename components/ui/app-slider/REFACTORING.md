## ✅ Что было сделано

### 1. Выделены переиспользуемые хуки

#### `useSliderGesture`
- Инкапсулирует логику Pan-жеста
- Поддерживает ограничения для range mode
- Универсален для любого ползунка

#### `useTooltipPosition`
- Расчет позиции тултипа с учетом границ экрана
- Настраиваемый отступ от краев
- Работает в worklet на UI thread

### 2. Создан подкомпонент SliderThumb

- **Инкапсулирует**: тултип + ползунок + жесты
- **Убирает дублирование**: для single и range используется один компонент
- **Упрощает**: главный компонент просто рендерит N thumbs

### 3. Вынесены типы

- Централизованное место для всех типов
- Легко найти и обновить интерфейсы
- Переиспользование в других компонентах

### 4. Добавлена документация

- README с архитектурой и примерами
- Комментарии JSDoc в коде
- Понятная структура для новых разработчиков

## 📝 Миграция

Не требуется! Компонент используется так же:

```typescript
import { AppSlider } from '@/components/ui/app-slider';

<AppSlider
  mode="range"
  min={0}
  max={2000}
  step={100}
  valueFrom={startPrice}
  valueTo={endPrice}
  onRangeChange={(from, to) => {
    setStartPrice(from);
    setEndPrice(to);
  }}
  formatValue={(value) => `${value}₽`}
/>
```
